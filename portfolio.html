Galaxy Blitz

Play Game

Project Description

Individual 2D Endless JavaScript game made with a lightweight JS library(Kenta Cho's crisp-game-lib). Focuses on single-button input and simple graphics primitives to narrow design parameters. This main focus of this game is it's progression system in which the player can power up their ship in order to go into "Blitz Mode".

Background

This project was my individual game which of a 3-game assignment where students needed to make : one game individually, one game in collaboration with one other collective member, and one game in collaboration with two or more collective members.
